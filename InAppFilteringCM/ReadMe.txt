    InAppFilteringCM (IAF) : Project overview

1. Project purpose.
	1.1.	Gets loaded by telshell.exe
	1.2.	Hooks IAF::SearchInvoked function and takes control over Search key action.
	
2. Architecture.
	2.1.	DLL hook is set up using Cloak Filter.
	2.2.	IAF calls old library via cloaking.
